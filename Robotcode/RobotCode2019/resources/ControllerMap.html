FRC 1736 Robot Casserole

These are the controls for the FRC 1736 robot.

Driver - Xbox Controller

Input Function Description
LX Axis
LY Axis Fwd/Rev Command Push up to make the robot move forward, down to make it run in reverse.
LStick Push
RX Axis Rotate Command Push left to make the robot spin counter-clockwise, right to spin clockwise
RY Axis
RStick Push
L Bumper Reverse Drive Enable Press and hold to make the back of the robot the "front"
R Bumper Snail Speed Enable Hold to move in "slow-mode", with finer control over drivetrain motion.
R Trigger Axis
L Trigger Axis
B Button
A Button Angle Lock Hold to enable closed-loop locking of pose angle. Useful for defense or accurate placement.
Y Button
X Button
POV Hat UP
POV Hat Right
POV Hat Left
POV Hat DN
Back Button Compressor Disable Tap to inhibit the pneumatic compressor from running
Start Button Compressor Enable Tap to enable the pneumatic compressor to run

Operator - Xbox Controller

Input Function Description
LX Axis
LY Axis Arm Position Manual Adjust Move up or down to manually move the arm position
LStick Push
RX Axis
RY Axis Climber Release Push past 50% deflection to command the climber ramp to be released.
RStick Push
L Bumper Gamepiece Grab Pull in to grab a gamepiece
R Bumper Gamepiece Release Pull in to release the currently-held gamepiece
R Trigger Axis Intake Pull In Hold to run the intake in the "in" direction to pull in cargo
L Trigger Axis Intake Eject Hold to run the intake in the "out" direction to eject cargo
B Button
A Button Arm Pos Cargo Pickup Position Press to move the mechanisms into place to pick up cargo from the floor
Y Button Arm Pos Hatch Pickup Position Press to move the mechanisms into place to pick up a hatch from the loading station
X Button Auto Sequence Hold this button while holding a POV hat position to initiate the auto placement sequencer at the commanded height.
POV Hat UP Arm Pos Score High Press to command the arm to the high rocket goal position
POV Hat Right Arm Pos Score Middle Press to command the arm to the middle rocket goal position
POV Hat Left Arm Pos Score Middle Press to command the arm to the middle rocket goal position
POV Hat DN Arm Pos Score Low Press to command the arm to the low rocket goal or cargo ship position
Back Button Climber Release Enable Press and hold to enable deployment of climber ramp.
Start Button
